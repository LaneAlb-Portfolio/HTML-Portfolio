Digital Game Projects

Awesome Name Goes Here

Created in a group of 4 at Moorpark College for the Game Design Technologies I course, the assignment was to create a digital game which utilized the Lock and Key mechanic. Gameplay and a link to the final build of the game can be found below. A write-up about my specific contributions can be found here.

The player must traverse a hellish environment filled with dangers and tricks in order to find the key to unlock the final door. Each level has its own unique environment and represents different stages of the game. In the beginning a lost person must traverse a maze, but after completing this maze they are met with a brutal volcano with the sweet sense of home in the distance. But upon entry this home isn't quite as comforting as the player thought..

SPYCE

SPYCE is a digital card game created for the ARTG 80G course at UCSC.

The player is a spy pretending to be a chef at a restaurant, however other spies have also taken up a post in this establishment. In order to win each player must prove that they are a chef rather than a spy and complete dishes faster than others. Of course backstabbing is fair game in the world of secrecy.

Coding Projects

  • C++ Maze

    Create a maze using an input file with a fixed character size of 30x100 (30 rows, 100 cols). Read the maze to create it and have a "mouse" run through the maze using a self built pseudo-Stack structure. Create Bombs ('B') in the maze that will explode when the mouse hits them.

  • C++ Pong

    Create a game within c++ only using tools available to you and other tutorials. Created in a development team of 4 people including myself.

  • C++ Trees

    Read numbers from a file and create an order binary tree. Allows the user to add values to the tree and remove/delete values. The tree can be output in 3 ways preorder, inorder, or postorder.
